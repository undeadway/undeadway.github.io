# 1. 概述

《大运动会》是一款主打轻量化、强反馈、易上手的休闲竞技小游戏。

游戏以“趣味”为核心进行包装，将大众熟知的经典游戏机制进行解构与重组，以低门槛的休闲体验降低玩家的学习成本，同时通过魔性物理反馈与碎片化关卡设计，满足玩家随时随地“来一局”的娱乐需求。  
《大运动会》不追求硬核竞技，而是以“轻松一笑”为目标，试图给玩家一个突然袭击。  
比如在最后一刻靶标突然移动，从而导致失败。  
又或者明明位置没有对准，却有“守株待兔”似的好运。

单局时长控制在30-60秒：无需复杂准备，随时开启、随时结束，适配通勤、午休等碎片化场景。

### 2.1.2. 玩法说明

玩家需根据风向、距离、障碍物（移动的靶子）调整出手角度与力度。标枪在空中可触发“旋转加速”“精准命中弱点”等Buff，落地后还会产生夸张的物理形变与音效反馈，强化“一击入魂”的爽感。
