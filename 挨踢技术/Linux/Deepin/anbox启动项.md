# Anbox 启动项

```
[Desktop Entry]
Categories=System;
Comment=uengine 程序菜单
Encoding=UTF-8n
Exec=/usr/bin/uengine-launch.sh --package=org.anbox.appmgr --component=org.anbox.appmgr.AppViewActivity
Icon=anbox
MimeType=
Name=uengine 程序菜单
StartupWMClass=uengine 程序菜单
Terminal=false
Type=Application
```
