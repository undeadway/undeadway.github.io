# 稀疏数组

定义：
> 稀疏数组是指**索引不连续**，数组长度大于元素个数的数组 ，通俗地说就是 有空隙的数组。

一些实例：
```
// 构造函数声明一个没有元素的数组
var a = new Array(5);    // [empty × 5]

// 指定的索引值大于数组长度
var a = [];
a[5] = 4;                // [empty × 5, 4]

// 指定大于元素个数的数组长度
var a = [];
a.length = 5;            // [empty × 5]

// 数组直接量中省略值
var a = [0,,,,];         // [0, empty × 3]

// 删除数组元素
var a = [0, 1, 2, 3, 4];
delete a[4];             // [0, 1, 2, 3, empty]
```
