# 基本工作规范

1. 每天提交一次当天工作报告，具体的工作报告规范参考：[工作报告规范](./工作报告规范.md)；
2. 所有代码都需要进行审查，具体的审查规范参考：[代码审查规范](./代码审查规范.md)；
3. 所有代码都需要进行测试，具体的测试规范参考：[测试规范](./基本测试规范.md)；
4. 提交代码时，需要遵守代码的提交规范：[代码提交规范](./代码提交规范.md)；
